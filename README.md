# Sistem Informasi Tingkat Kesegaran Buah Mangga

Sistem informasi berbasis AI untuk mendeteksi dan mengklasifikasikan tingkat kesegaran buah mangga secara otomatis.

## Fitur Utama
- **Deteksi Kesegaran**: Menggunakan Computer Vision untuk menentukan apakah mangga segar, matang, atau busuk.
- **Analisis Kualitas**: Memberikan informasi detail mengenai kualitas buah berdasarkan parameter visual.
- **Dashboard Informatif**: Antarmuka pengguna yang intuitif untuk memantau data kesegaran buah.

## Teknologi
- **Backend**: Python / FastAPI / Go (Opsional)
- **Frontend**: React / JavaScript
- **Model AI**: TensorFlow / PyTorch / OpenCV
- **Database**: PostgreSQL / MongoDB

## Cara Penggunaan
1. Clone repositori ini.
2. Instal dependensi yang diperlukan.
3. Jalankan aplikasi menggunakan perintah `npm run dev` atau perintah yang sesuai.
